# TakeASnip

TakeASnip is a Django web application designed for developers to share, rate and comment on code snippets.
The platform supports collaboration, knowledge sharing and user engagement through a gamification system.

## Project Overview
TakeASnip allows users to publish reusable code snippets, categorize them and interact with content created by others.
The application includes user roles, security mechanisms and an administration panel.

---

## Key Features

### 1. User Registration and Authentication
- Users can register by providing a unique username and password
- Users can log in and log out of the system
- The system supports user roles:
  - **User**
  - **Administrator**
- Passwords are securely stored using appropriate security mechanisms

---

### 2. User and Role Management
- Administrators can manage users (e.g. assign and revoke roles)
- Each user has a profile containing:
  - experience points (XP)
  - rank
  - earned badges

---

### 3. Code Snippet Management
- Users can create, edit and delete their own code snippets
- Snippets include:
  - code content
  - description
  - technology tags
  - categories (e.g. algorithms, application templates, frameworks)
- Administrators have full CRUD access to all snippets

---

### 4. Rating and Comment System
- Users can upvote and downvote snippets and comments
- Users can add comments to snippets
- The system supports community feedback and content quality evaluation

---

### 5. Gamification System
- Users earn experience points (XP) for activities such as:
  - adding code snippets
  - rating content
  - commenting
- User ranks are assigned based on XP:
  - **Beginner**
  - **Advanced Coder**
  - **Master Coder**
- XP and ranks are visible on user profiles

---

### 6. Administrator Panel
- Full CRUD management for:
  - users
  - code snippets
- Role management for users
- Access to application statistics and user activity overview
